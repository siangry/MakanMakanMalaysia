# Front-End Web Application - MakanMakanMalaysia

A website to explore rich traditions of Malaysian cuisine.
The webpages included in this application are:
1. Home page: To provides a comprehensive introduction to Malaysian cuisine.
2. Recipes page: To display food recipes with different categories.
3. Cultural insights page: To display the history, unique ingredients, and interviews of Malaysia food experts.
4. Restaurant guide page: To display restaurant details, suggested menu, and map.
5. Community page: To have discussion in the community by uploading posts and leaving comments.
6. Personal blog page: To displays the user profile and upload blog posts.
7. About us page: To display goals, mission, vission, and team members.
8. Login and signup pages: To create an account or log in.

## Tech Stack
- HTML
- CSS
- JavaScript
- Bootstrap
- RESTful API
